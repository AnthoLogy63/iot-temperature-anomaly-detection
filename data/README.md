# data/

Contiene los datasets de entrada para el entrenamiento y evaluación de los modelos.

## Archivos

- `iot_sensor.csv` — Datos de sensores IoT (etiquetados o no etiquetados)
- `clean_unlabeled_autoencoder.csv` — Dataset sin etiquetas para preentrenamiento
- `synthetic_iot_dataset_challenging.csv` — Dataset sintético con patrones complejos
